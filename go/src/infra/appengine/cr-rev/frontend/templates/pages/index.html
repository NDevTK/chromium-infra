Welcome to cr-rev!

Usage:

  • crrev.com/<git commit position, git sha, rietveld issue>
  • crrev.com/c/<gerrit CL number> (chromium-review)
  • crrev.com/i/<gerrit CL number> (chrome-internal-review)

You can append subpaths to each short rev too to navigate to subdirs in gitiles. e.g. crrev.com/<git sha1>/tools/OWNERS
